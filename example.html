This drawing is a vector drawing, rather than a paint-style drawing. All drawing items that you place can be altered or deleted, you do not just set pixels.

The drawing normally starts at a size of 1000 by 1000 units and you can zoom and pan around.

You can place drawing items by pressing the relevant button for the type of drawing item images here. To place Rectangles and Circles, place the cursor and then drag to set diagonal corners. To place Text click to place the centre of the text and you will be prompted for the text. To draw a Freehand line trace the shape. To draw a straight Line place the cursor for the start point and then drag to place the end point.

When you place a drawing item it will be created with a default line colour, fill colour, line thickness and font size, as appropriate. You can change the default settings before you place an item by selecting the Colors button. Alternatively, you can change the properties of a drawing item after placing it.

To alter an existing drawing item, select the Select button. You can then select a drawing item by clicking on it. This will bring up a dialog that allows you to change colour and other properties. The dialog also allows you to delete an item.

You can also select an item and drag it to change its position.

You can change the "Z" rating of an item to move it in front of or behind other items.

You can group items together by pressing the Group Select button. Drag to select a rectangle and all items that are wholly within the rectangle will be includedin the group. Once you have a group, if you select and drag one item in the group then all items are dragged together. If you select an item that is in a group then you can remove that item from the group or remove all items from the group.

You can export a copy of the drawing locally by pressing the Export button. This opens another browser window containing the drawing as an image. You can then save this locally to your computer.
